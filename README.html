Plugin Sample by Features / Amostra por feição

Este plugin elabora um plano de amostragem orientado por feição. O objetivo é permitir a realizição de inspeções de qualidade para a aprovação ou reprovação de um conjunto de dados espaciais, segundo níveis de conformidade desejados. 

Funcionalidades

  1. O tamanho da popução (N) é baseado na quantidade de registros da camada desejada.  
  2. O tamanho da amostra (n) é obtido com base nas tabelas padronizadas da ISO 2859.
  3. O usuário define quais serão os critérios avaliados.
  4. Este plugin é recomendado para aferição da acurácia temática do conjunto de dados. 

Notes:

  • O plugin é baseado no Manual Técnico de Geociências de Avaliação da Qualidade de Dados Geoespaciais do IBGE. 
  • Mais informações deste plugin estão disponíveis em: https://github.com/alxcart/SampleByFeatures/wiki
